Privacy Policy

Last Updated: 10 Nov 2025

AutoPDF Collector (“the Extension”) is designed to help users collect webpage content, organize notes, generate summaries, and export documents. We respect your privacy and are committed to keeping your data secure.

1. Information We Collect

AutoPDF Collector does not collect any personal data unless explicitly provided by the user.

Data stored locally on your device:

  • Snippets you capture from websites
  • Preferences (theme, font style, zoom, settings)
  • Saved projects (max 5)
  • AI configuration (API keys you manually enter — stored locally only)
  • Premium plan status & expiry (stored locally)

No data is sent to our servers because we do not run any servers.

2. AI Services (Optional)

If you choose to use AI features (Summarize, Chat With Snippet), the Extension may send the selected text to:

  • OpenAI API
  • Google Gemini API
  • Hugging Face Inference API

These actions occur only after you enter your own API key and manually trigger an AI request. Your API key is stored locally and never shared with us or any third party.

3. Payment Processing

AutoPDF Collector uses Razorpay payment links. We do not collect or store any billing or payment information.

Razorpay processes payments independently and securely. After payment, Razorpay redirects you to a **success page** that sends a message to the extension to activate Premium.

4. Permissions We Use

The Extension requests Chrome-specific permissions solely to function:

  • storage — to save your snippets and settings locally
  • activeTab — to capture selected content
  • scripting — to insert content scripts
  • contextMenus — for “Add to AutoPDF Collector” button
  • downloads — to export PDF/HTML/JSON files

We never access browsing history, cookies, passwords, or personal accounts.

5. How We Handle Your Data

All user-generated content stays on your device unless you export it yourself.

We do NOT:

  • Collect analytics
  • Share or sell your data
  • Store any snippets on a server
  • Track user activity across websites

6. Third-Party Integrations

Some features use optional third-party APIs and libraries:

  • OpenAI / Gemini / HuggingFace APIs for summaries (optional)
  • PDF creation libraries (local only, no data sent outside)
  • Razorpay for payments

Their respective privacy policies apply only when using those features voluntarily.

7. Children's Privacy

This extension is not intended for children under the age of 13.

8. Security

We take security seriously. All data remains inside Chrome’s secure storage. No data leaves your device without your action.

9. Changes to This Policy

We may update this policy as features evolve. The latest version will always be available on this page.
